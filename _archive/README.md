# _archive/

Archivordner für historische Spuren — Dokumente, die nicht mehr aktiv sind, aber als Audit-Spur erhalten bleiben sollen. Konvention angelehnt an Lesevault-Archiv:

- Prefix `ARCHIVED_` vor dem Original-Dateinamen.
- Archivierungsvermerk am Anfang des Dokuments: `# ARCHIVED on YYYY-MM-DD - Reason`.
- Links in anderen Dateien aktualisieren oder als historisch markieren.
- Nie löschen, immer archivieren — Ausnahme: vollständig in andere Dokumente extrahierte Inhalte können gelöscht werden (Quelldokumentation in LOG).

Schreibhoheit beim Strukturarbeiter (#2). Aktuell leer — wird befüllt, sobald erste Dokumente ihre aktive Phase verlassen (z.B. Bootstrap-Phase-Dokumente nach Phase-B-Konsolidierung).
